Creation of the automatical documentation for package

1) Downloading packages
  - devtools -> contain function document()
  - roxygen2 -> Create automatical documentation
  - Rtools -> https://cran.r-project.org/bin/windows/Rtools/

2) Heading with the template with Template_function.R
Think about #' a the beginning of line

Different section : @xxx
Each section can be in different order except the fisrt two sections
(short resume of function and the explanation of the function)

Problem : % must be replace by \%

Tips :
\cr -> text wrapping
\itemize -> bullet point
    example :
    @return zoo
    \itemize{
    \item Wet
    \item Normal
    \item Dry
    }
You can include different itemize

More information
http://r-pkgs.had.co.nz/man.html
https://bookdown.org/rdpeng/RProgDA/documentation.html
https://cran.r-project.org/web/packages/roxygen2/vignettes/formatting.html

3) Creation of documentation
Step 1 : Function with the new heading
Step 2 : Delete all files in man folder
Step 3 : Use document function : document()
Step 4 : Install and restart package in the "build" tab, your package will be
         check and install in your Rstudio session (Ctrl+Shift+B).
Step 5 : check the result with the "help" tab with your different function.
